# openLog icon - white rounded corrected

This pack fixes the corner problem.

Design:
- White/light Variant 1 icon
- Transparent outside rounded corners
- No dark tile
- No square opaque corners
- Same visual footprint as a normal desktop app icon

Use:
- Compose runtime/window icon: `compose-resources/icons/openlog.png`
- macOS native package: `native/openlog.icns`
- Windows native package: `native/openlog.ico`
- Linux native package: `native/openlog-linux-1024.png`

Important:
The PNG is RGBA. The corners are transparent; the app tile itself remains white.
